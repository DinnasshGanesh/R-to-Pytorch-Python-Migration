# R-to-Pytorch-Python-Migration

### Research Objective
A comparative study migrating a traditional hydrological water-balance model (R/Shiny) into a high-performance **Physics-Informed Neural Network (PINN)** framework using **PyTorch**.

---

## 📁 Repository Structure
* **01_R_Prototype:** Original R-Shiny implementation of the green roof storage tank dimensioning tool.
* **02_PyTorch_Migration:** Active development of the tensor-based model, utilizing Autograd for mass-balance constraints.
* **03_Technical_Manual:** A Jupyter-based guide documenting the mathematical transition from R to Python.

---

## 🏗️ Technical Highlights
* **Domain:** Urban Hydrology & Green Infrastructure.
* **Tech Stack:** R (Shiny), Python (PyTorch), Google Colab.
* **Scientific Method:** Implementing Hargreaves PET equations as differentiable layers.
